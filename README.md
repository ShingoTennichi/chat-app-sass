# Real-Time Chat App(Websocket)

- JavaScript
- Node.js
- HTML
- SCSS(EBM)

## Commands
```sh
# install required packages
npm i

# run database server
node ./backend/.

# run websocket server
node ./backend/websocket/.

# ====================================
# for dev
# run database server
node --watch ./backend/.

# run websocket server
node --watch ./backend/websocket/.

# compile sass to css
cd frontend
npm run sass-dev
```
